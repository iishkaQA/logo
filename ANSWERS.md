1. Нужно ли было писать свой генератор фамилий?

Нет. Использование Faker значительно упрощает задачу, поскольку библиотека уже содержит большие наборы реалистичных данных и готовые методы для их генерации.

2. Какая библиотека больше подходит для автотестов?

Faker. Она позволяет быстро генерировать случайные имена, телефоны, адреса и другие данные, необходимые для тестирования приложений. JFiglet используется только для оформления текста в консоли.

3. Как JAR-файлы этих библиотек попали в проект?

Через Gradle. После добавления зависимостей в build.gradle система сборки автоматически скачивает необходимые JAR-файлы из Maven Central и подключает их к проекту.
